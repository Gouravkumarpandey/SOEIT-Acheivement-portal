Use Case Specifications

Authentication Module

Use Case ID Use Case Name Actor(s) Precondition Postcondition
UC-AUTH-01 Register Account Public User User not registered New account created
UC-AUTH-02 Login All Users Valid credentials, CAPTCHA User authenticated, JWT issued
UC-AUTH-03 Forgot Password Public User Valid registered email Reset link sent via email
UC-AUTH-04 Reset Password Public User Valid reset token Password updated
UC-AUTH-05 Edit Profile Authenticated User User logged in Profile information updated
UC-AUTH-06 Change Password Authenticated User User logged in, knows current password Password changed

Achievement Module

Use Case ID Use Case Name Actor(s) Precondition Postcondition
UC-ACH-01 Submit Achievement Student Student logged in Achievement created with status "pending"
UC-ACH-02 Upload Proof Files Student Submitting achievement Files uploaded and compressed
UC-ACH-03 View My Achievements Student Student logged in List of achievements displayed
UC-ACH-04 Edit Achievement Student Achievement exists, status is pending Achievement updated
UC-ACH-05 Review Achievement Faculty, Admin Pending achievements exist Achievement reviewed
UC-ACH-06 Approve Achievement Faculty, Admin Achievement reviewed Status changed to "approved", points added
UC-ACH-07 Reject Achievement Faculty, Admin Achievement reviewed Status changed to "rejected", notification sent

UC-CRS-02 Update Progress Student Course exists Progress percentage updated
